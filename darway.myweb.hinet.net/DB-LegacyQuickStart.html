※下載 Win32 DooM Legacy v1.42 ( 發佈日期2004- 04-17 )

※設定好的檔案：config.cfg (用滑鼠玩) 與 launcher.cfg (請【按此下載】...解壓縮之後的檔案放到 Legacy 目錄底下)

●按"LAUNCHER.EXE"後再進入”Game Mode..."選擇想要玩的地圖：

●按"Launch"啟動遊戲：

●設定顯示方式(如果不行玩的話再設......)

 

※手動設定(如果自己想設看看的話.......)：

●第一次按"LAUNCHER.EXE"啟動時的畫面設定：

1.設定"Legacy.exe"所在路徑。

2.設定四大地圖檔所在路徑和選擇想要玩的地圖。

3.設定"設定檔"名為"config"。

●按"Setup..."設定顯示方式：

●進入"Setup..."後設定顯示方式：

1.Software：軟體模擬(無3D加強，跑3D加強很吃力時用)

2.3Dfx Glide：3D加強(voodoo系列)

3.OpenGL：3D加強(nVidia的3d顯示卡(TNT或GeForce系列))

4.miniGL：3D加強(voodoo系列或其它的顯示卡)

●按"Launch"啟動遊戲：

●進入遊戲後(如果下載了設定好的檔案，以下就不用設定了)......

1.按鍵"F11"四下，使亮度成為最高比較清楚

2.按鍵"ESC"切換→OPTIONS→ALWAYS RUN:ON(總是跑功能開啟)

3.OPTIONS→MOUSE OPTIONS→ALWAYS MOUSELOOK:ON(使用滑鼠描準)

4.OPTIONS→CROSSHAIR:CROSS(十字準心)、ANGLE(角準心)、POINT(點準心)

5.OPTIONS→SOUND VOLUME...→SFX VOLUME(音效大小)；MUSIC VOLUME(音樂大小)

6.OPTIONS→VIDEO OPTIONS...→3D CARD OPTIONS...→FOG...→FOG:ON(濾鏡功能開啟)
；FOG COLOR:FFFFFF(白色；變比較亮)；FOG DENSITY:100(愈大濾鏡愈強)

※外掛地圖 PWAD 部分是在 Single Map 裡設定，請參考【開 server 設定-步驟５】

※用鍵盤玩(legacy預設值)："↑"(前), "↓"(後), "←"(左), "→"(右), "Ctrl"(開火),"Space"(開門),
"Shift"＋方向鍵(跑), "Alt"＋方向鍵(側走), "Shift"＋"Alt"＋方向鍵(側跑),
"TAB"(地圖), "T"(傳訊息), "/"(跳), "Page Up"(向上看), "Page Down"(向下看), "End"(視眼置中)

※用滑鼠玩(建議)："左鍵"(前), "D"(後), "S"(左), "C"(右), "右鍵"(開火),"Space"(開門),
"TAB"(地圖), "T"(傳訊息), "Alt"(跳)

◎滑鼠按鍵設定從朋友UAC(很強)所學來的^^"

Multiplayer 連線教學

進入程式 Doom Legacy Lanucher 時選擇下面的 Multiplayer：


【如果是由您當 Server 請進行以下設定】：

  • 步驟１：選擇 Server。
  • 步驟２：選擇 Tcp/Ip。
  • 步驟３：設定遊戲人數，設定幾人都沒關係，但注意一個原則就是不要設定超過實際遊戲人數， 因為經過測試除了開 Server 的人以外其他的 Client 端均可自由的 Join 和 Leave Game， 所以設定 1 人就可以了，遊戲進行的最大人數上限為該地圖的最大再生點數目， 也就是說遊戲進行的最大人數由地圖決定。
  • 步驟４：設定遊戲的進行狀況 ( 如 Skill，Game mode，Team play .... )。
  • 步驟５：如果要外掛其它的地圖，請按"+"鈕選地圖(最好其它的地圖都放在 Legacy 的目錄下)， 然後把檔名兩旁的""去掉，不然會找不到檔案（算是 Legacy 的 bug 吧）。
  • 附　註：如果有掛兩個以上的 .wad 或 .deh，第一個會抓不到所以亂打一個檔名，例："xxx" 或使用參數的方式打在 Additional options 的框裡，例：-file "地圖檔1" "地圖檔2"
  • 步驟６：按下確定即進入等待，等連線人數達到設定人數後 Game 隨即展開


【如果當您是 Client 請進行以下設定】：

  • 步驟１：選擇 Client
  • 步驟２：選擇 Tcp/Ip
  • 步驟３：選擇 Connect to specified server , 並在下面打入 Server 端的 IP
  • 步驟４：按下確定即進入等待，等連線人數達到設定人數後 Game 隨即展開
  • 附　注：假如地圖裡的怪物太多造成跳出遊戲，跳出說記憶體不足的情形，可用參數-mb來加記憶體，例：-mb 64
